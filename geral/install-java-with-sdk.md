# Install java with SDK

[INICIO DA README](../README.md)

> **Documentation:** https://sdkman.io/install

```bash
curl -s "https://get.sdkman.io" | bash
```

```bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

```bash
sdk version
```

```bash
sdk list java
```

```bash
sdk install java 11.0.11-open
```
